4. Rebuild all containers: `cd ~/talkflow-v2 && sudo docker compose build`
5. Launch: `sudo docker compose up -d`
6. Verify worker: `sudo docker logs -f talkflow-worker`

### Phase 3: Enable GPU (~5 min)
1. Configure NVIDIA Container Toolkit:
   ```bash
   sudo nvidia-ctk runtime configure --runtime=docker
   sudo systemctl restart docker
   ```
2. Restart containers: `sudo docker compose up -d`
3. Verify GPU: `sudo docker exec talkflow-worker nvidia-smi`
