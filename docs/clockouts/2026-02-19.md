## How to Run (Local Setup)

### Prerequisites
- **FFmpeg**: Required for video encoding (`brew install ffmpeg`).
- **macOS 'say'**: Native (used for the TTS stub).
- **Python 3.10+** & **Node.js 18+**.

### Run Commands
```bash
# Start Backend
cd api_server && python -m app.main

# Start Frontend
cd frontend && npm run dev
```

## How to Test
- **Audio-to-Video**: Upload a `.wav` file in the "Generate" view. Verify the resulting MP4 has that audio.
- **Text-to-Video**: Write a script and generate. Verify the video duration matches the spoken text length.
- **Gallery**: Confirm that a Shakespeare video shows a Shakespeare thumbnail and plays in the modal.

## Known Issues / Limitations
- **LipSync Movement**: While audio is real and synced, the visual is still a static loop of the avatar image (MuseTalk binary integration pending).
- **Polling Latency**: 10s polling is sufficient for local dev but might feel slow for very short generations.
