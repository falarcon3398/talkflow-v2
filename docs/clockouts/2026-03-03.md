## 3) Technical Debt & Gotchas
- **Signaling Order:** `aiortc` is sensitive to the order of operations. Adding tracks *before* setting the remote description for a browser-initiated offer can lead to "transceiver pollution" (duplicate entries).
- **Reloading:** `StatReload` (uvicorn) sometimes misses changes in nested `live_call` modules if the server isn't hard-restarted after signaling changes.
